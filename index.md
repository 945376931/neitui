### 如何添加内推

1. 克隆本项目
2. 在 company.txt 中添加公司名，自定义，以下记作 cmp
3. 添加 logo 到 assets 文件夹中，并命名为 logo-cmp.jpg
4. 添加 cmp 文件到 infos 文件夹中，文件内容为 markdown 格式的内推信息
5. 运行 gen.sh 生成 README
6. 解决冲突，发起PR
7. 将内推人工号，姓名，待合并 PR 发送至 sinksmell@163.com
